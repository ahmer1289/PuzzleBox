# PuzzleBox

**PuzzleBox** is an immersive escape-room style game where players solve riddles, interact with objects, and unlock secrets to escape fully designed rooms. Each room is packed with challenging puzzles, interactive mechanisms, and door systems that test your logic, observation, and problem-solving skills.

## Gameplay

- Explore fully built 3D rooms with hidden objects and clues.  
- Solve riddles and puzzles to unlock doors and progress.  
- Interact with objects in the environment to reveal secrets or trigger mechanisms.  
- Every room presents a unique challenge that tests your observation and logic skills.

## Features

- Fully built interactive rooms  
- Multiple puzzles and riddles per room  
- Interactive mechanisms, door systems, and triggers  
- Object interaction for clues, keys, and hidden items  
- Immersive environment design for an escape-room experience  
- Modular and extendable for adding new rooms and puzzles
